# Taskflow-Gremio

**# 📜 TaskFlow: Gremio Edition

Sistema de gestión de misiones desarrollado para el bootcamp de desarrollo web. Una aplicación robusta con estética de tablón de anuncios de fantasía.

## 🚀 Demo
Puedes ver la aplicación en vivo aquí: taskflow-gremio.vercel.app

## ✨ Características
- **Gestión de Misiones:** Crear, editar, completar y eliminar encargos.
- **Filtrado Avanzado:** Clasificación por Rangos (D-S) y Categorías (Caza, Escolta, etc.).
- **Estado del Gremio:** Panel de estadísticas en tiempo real.
- **Persistencia:** Guardado automático en `LocalStorage`.
- **Modo Oscuro:** Interfaz adaptable con persistencia de preferencia.

## 🛠️ Tecnologías
- **HTML5:** Estructura semántica.
- **Tailwind CSS:** Diseño responsive y configuración personalizada.
- **JavaScript (Vanilla):** Lógica de estado, manipulación del DOM y LocalStorage.

## 🧪 Pruebas Realizadas (Paso 10)
1. **Carga vacía:** El sistema muestra el panel de estadísticas en cero correctamente.
2. **Persistencia:** Se verificó que al recargar la página las misiones y el modo oscuro se mantienen.
3. **Validación:** No se permiten misiones sin título.
4. **Responsive:** Probado en iPhone 13 y escritorio.**
